Day 5 — Create VNet via Azure Portal (nautilus)

Task summary

Create a Virtual Network `nautilus-vnet` in `eastus` using the Azure Portal. The VNet must use the IPv4 CIDR `192.168.0.0/24`.

Files in this folder

- `QUESTION.md` — Task description and acceptance criteria (Portal-first)
- `ANSWERS.md` — Portal steps + automation (Azure CLI) and verification
- `create_vnet_portal.sh` — Small script to reproduce the Portal-created VNet via Azure CLI
