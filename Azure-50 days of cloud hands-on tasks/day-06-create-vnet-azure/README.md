Day 6 — Create VNet and Subnet (xfusion)

Task summary

Create a Virtual Network `xfusion-vnet` and a subnet `xfusion-subnet` in `eastus`. The VNet must use `10.0.0.0/16` and the subnet must use a valid, non-overlapping prefix inside that space (suggested `10.0.0.0/24`).

Files in this folder

- `QUESTION.md` — Task description and acceptance criteria
- `ANSWERS.md` — Example solutions, verification commands, and conceptual notes about subnets
- `create_vnet.sh` — Small script to create RG, VNet and subnet using Azure CLI
